## Bu branch ne içeriyor?

Bu branch, **RookieVerse’un yedeği değildir.**

RookieVerse reposuna yanlışlıkla **Faaliyet Raporlama Sistemi** kodları pushlandığı için (ör. `index.php` gibi dosyalar üzerine yazıldı), iki kod tabanını da kaybetmemek adına şu ayrım yapılmıştır:

- **backup** branch: Faaliyet Raporlama Sistemi kodları (koruma amaçlı)
- **main** branch: RookieVerse’un asıl kodları (restore edilmiş)
